# 코딩테스트 스터디
# 🗂️ Repository 폴더 구조
```
week@/BOJ_문제번호/문제이름_본인이름.언어
```
- 예시: `week1/BOJ_4673/셀프 넘버_김선진.py`

# ✅ commit 규칙
- commit 메시지: [BOJ] 문제번호_문제이름 / 난이도 / 문제풀이성공여부
```
git commit -m "[BOJ] 4673_셀프 넘버 / 실버 5 / O"
git commit -m "[BOJ] 4673_셀프 넘버 / 실버 5 / X"
```
